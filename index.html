About SIKPRO

SIKPRO is a desktop-based Statistical Process Control (SPC) application developed to assist users in performing quality control analysis through control charts, automatic calculations, graphical visualization, and Nelson Rules interpretation.

The application supports I-MR Chart, X̄-R Chart, X̄-S Chart, and several additional quality tools to support learning and implementation of SPC.
