SIKPRO v1.0

Thank you for downloading SIKPRO.

SIKPRO is a desktop-based Statistical Process Control (SPC) application designed to assist quality control analysis through control charts, Nelson Rules interpretation, and quality improvement tools.

SYSTEM REQUIREMENTS

✓ Windows 10 / Windows 11
✓ Minimum 4 GB RAM
✓ Minimum 500 MB Free Storage
✓ Recommended Screen Resolution: 1366 × 768 or higher

INSTALLATION GUIDE

Run the installer:

SetupSIKPRO_v1.0.exe

If Windows Defender displays a warning:

More Info
→ Run Anyway

Follow the installation wizard:

Next
→ Install
→ Finish

Once installation is complete, SIKPRO can be launched from:

• Desktop Shortcut
• Start Menu

FEATURES

CONTROL CHARTS

• I Chart
• I-MR Chart
• X̄-R Chart
• X̄-S Chart

NELSON RULES

• Automatic process violation detection
• Process stability interpretation
• Quality monitoring support

QUALITY TOOLS

• Histogram
• Pareto Diagram
• Fishbone Diagram
• Scatter Plot

PROJECT MANAGEMENT

• Save Project
• Open Project
• Data Management

TROUBLESHOOTING

Application does not start

• Run as Administrator
• Reinstall the application
• Verify antivirus software is not blocking SIKPRO

Windows Defender Warning

• Click "More Info"
• Click "Run Anyway"

Chart or image not displayed

• Reinstall the application
• Ensure installation completed successfully
